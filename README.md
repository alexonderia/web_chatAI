# Web ChatAI

Одностраничное приложение на React + Vite + TypeScript + Material UI, созданное по мотивам макета для быстрого запуска диалогового ассистента.

## Ключевые возможности интерфейса
- Верхняя панель с логотипом, ссылкой на регистрацию и акцентной кнопкой входа.
- Центральный вопрос "Чем помочь?" и крупная строка ввода с иконками микрофона и поиска.
- Чипы с подсказками под строкой ввода для примеров запросов.
- Карточки преимуществ: контекстные подсказки, голосовой ввод, быстрый старт через регистрацию.
- Цветовая схема с акцентами синего, голубого и белого, мягкие закругления и тени.

## Стек
- React 18, TypeScript, Vite
- Material UI 5, Emotion
- Настроенная тема (акцентные синие оттенки), Roboto как базовый шрифт

## Скрипты
- `npm run dev` — запуск дев-сервера Vite.
- `npm run build` — сборка production-бандла.
- `npm run preview` — предпросмотр собранного приложения.
- `npm run lint` — базовая проверка ESLint.

## Docker
Проект собирается полностью в Docker, включая установку зависимостей:

```bash
# Сборка образа
docker build -t web-chatai .

# Запуск контейнера
docker run -p 8080:80 web-chatai
```

Приложение будет доступно на `http://localhost:8080`.

## Разработка
1. Установите зависимости: `npm install`.
2. Запустите локально: `npm run dev` (по умолчанию порт 5173).
3. Настройки темы и цветовой схемы находятся в `src/theme.ts`, глобальные стили — в `src/styles.css`.

## Чистота проекта
- Разделение UI на небольшие компоненты.
- Единая тема Material UI и переиспользуемые цвета.
- Линтинг с рекомендациями TypeScript и React Hooks.
